1ï¸âƒ£ Connecting Frontend and Backend Concepts
Notes

The frontend (HTML/CSS/JS/React/Vue) communicates with a Java backend (Spring Boot) via HTTP requests.

The backend exposes REST endpoints such as /api/users.

Data exchanged is usually in JSON format.

Frontend uses fetch() or Axios to call backend APIs.

Java Backend Example (Spring Boot Controller)
@RestController
@RequestMapping("/api")
public class MessageController {

    @GetMapping("/message")
    public Map<String, String> getMessage() {
        return Map.of("message", "Hello from Java backend!");
    }
}

Frontend Example
fetch("http://localhost:8080/api/message")
  .then(res => res.json())
  .then(data => console.log(data.message));

2ï¸âƒ£ REST API Basics & HTTP Methods
Notes

REST API uses standard HTTP methods:

Method	Purpose	Example Endpoint
GET	Retrieve data	/api/users
POST	Create new data	/api/users
PUT	Update data	/api/users/{id}
DELETE	Remove data	/api/users/{id}
Java REST Examples
GET
@GetMapping("/users")
public List<String> getUsers() {
    return List.of("Alice", "Bob");
}

POST
@PostMapping("/users")
public User createUser(@RequestBody User user) {
    return user; // returns created user
}

PUT
@PutMapping("/users/{id}")
public String updateUser(@PathVariable int id, @RequestBody User user) {
    return "Updated user " + id;
}

DELETE
@DeleteMapping("/users/{id}")
public String deleteUser(@PathVariable int id) {
    return "Deleted user " + id;
}

4ï¸âƒ£ Async Programming with Callbacks (Java)
Notes

Java backend handles async operations using:

Threads

CompletableFuture

@Async in Spring Boot

Java does not use callbacks like JavaScript, but similar behavior is provided with futures/promises.

Async Example (Spring Boot)

Enable Async

@EnableAsync
@SpringBootApplication
public class App {}


Async Service

@Service
public class AsyncService {

    @Async
    public CompletableFuture<String> loadData() throws InterruptedException {
        Thread.sleep(1000);
        return CompletableFuture.completedFuture("Async data loaded!");
    }
}


Controller

@Autowired
private AsyncService asyncService;

@GetMapping("/async")
public CompletableFuture<String> asyncExample() throws Exception {
    return asyncService.loadData();
}

package com.weatherapp.controller;

import com.weatherapp.service.WeatherService;
import org.springframework.web.bind.annotation.*;

@RestController
@RequestMapping("/api/weather")
@CrossOrigin("*")
public class WeatherController {

    private final WeatherService service;

    public WeatherController(WeatherService service) {
        this.service = service;
    }

    @GetMapping("/{city}")
    public String getWeather(@PathVariable String city) {
        return service.getWeather(city);
    }
}

ğŸ“Œ 4. WeatherService.java
package com.weatherapp.service;

import org.springframework.beans.factory.annotation.Value;
import org.springframework.stereotype.Service;
import org.springframework.web.client.RestTemplate;

@Service
public class WeatherService {

    @Value("${weather.api.key}")
    private String apiKey;

    @Value("${weather.api.url}")
    private String apiUrl;

    public String getWeather(String city) {
        String url = apiUrl + "?q=" + city + "&appid=" + apiKey + "&units=metric";

        RestTemplate template = new RestTemplate();
        return template.getForObject(url, String.class);
    }
}

            document.getElementById("error").innerText = "";
            document.getElementById("city-name").innerText = data.name;
            document.getElementById("temp").innerText = data.main.temp + " Â°C";
            document.getElementById("desc").innerText = data.weather[0].description;

            document.getElementById("weather-box").classList.remove("hidden");

            document.getElementById("error").innerText = "Error fetching weather!";
