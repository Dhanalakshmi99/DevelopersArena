(GitHub Actions / GitLab CI)
Goal

Automatically:

Build

Scan

CI/CD Stages
Code â†’ Build â†’ Test â†’ Security Scan â†’ Push Image â†’ Deploy

Example: GitHub Actions Pipeline
name: CI-CD

on:
  push:
    branches: [main]

jobs:
  build-test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Build image
        run: docker build -t myapp:${{ github.sha }} .
      - name: Run tests
        run: docker run myapp:${{ github.sha }} npm test

  push-image:
    needs: build-test
    runs-on: ubuntu-latest
    steps:
      - run: docker tag myapp:${{ github.sha }} myrepo/myapp:latest
      - run: docker push myrepo/myapp:latest

  deploy:
    needs: push-image
    runs-on: ubuntu-latest
    steps:
      - run: helm upgrade --install myapp helm/

Key Notes

CI = build, test, scan

CD = deploy to Kubernetes

Pipelines should fail fast
